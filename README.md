# Commit Message Helper

一个自动生成 Git 提交信息的 VS Code 扩展。

## 功能特性

- 🤖 智能分析代码变更
- 📝 自动生成有意义的提交信息
- ⚡ 支持多种编程语言
- 🎯 一键提交操作
- 🔧 可配置的生成规则

## 安装

1. 在 VS Code 扩展市场中搜索 "Commit Message Helper"
2. 点击安装
3. 重新加载 VS Code

## 使用方法

### 方法一：命令面板
1. 按下 `Ctrl+Shift+P` (Windows/Linux) 或 `Cmd+Shift+P` (Mac)
2. 输入 "生成提交信息"
3. 选择生成的提交信息并确认

### 方法二：快捷键
- 使用 `Ctrl+Shift+G` (Windows/Linux) 或 `Cmd+Shift+G` (Mac) 快速生成

### 方法三：源代码管理面板
1. 点击 VS Code 左侧的源代码管理图标
2. 在标题栏中找到提交按钮旁边的生成图标

## 配置

在 VS Code 设置中搜索 "Commit Message Helper" 进行配置：

- `commitMessageHelper.aiModel`: 选择生成模型（规则基础/AI增强）
- `commitMessageHelper.autoSuggest`: 是否自动建议提交信息

## 支持的文件类型

- JavaScript/TypeScript (.js, .ts, .jsx, .tsx)
- Python (.py)
- Java (.java)
- C++ (.cpp, .h)
- HTML/CSS (.html, .css, .scss)
- 配置文件 (.json, .yml, .yaml)
- 文档文件 (.md)
